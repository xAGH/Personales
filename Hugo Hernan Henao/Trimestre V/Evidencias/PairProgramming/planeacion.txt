1. Requisitos.
	1.1. Documentar el programa.
	1.2. El programa debe permitir evaluar x numero de listas del mismo tamaño, estas listas deben tener un nombre.
	1.3. El programa debe calcular la desviación media de cada una de las listas y al final del programa, listarlas
		 en orden descendente(De menor a mayor).
    1.4. La cantidad de listas y su contenido debe ser ingresado por teclado.
    1.5. Los números ingresados deben ser números reales.
    1.6. La aplicación será en consola.
    1.7. La aplicación debe tener un menú con al menos una opción para limpiar la pantalla.
    1.8. No se deberán usar funciones o programas fuera de las mismas propiedades de las listas en el lenguaje de 
    	 programación.
    1.9. El programa deberá ser desarrollado en Python.

2. Planeación.
	2.1. Averiguar el SO donde está corriendo el programa.
	2.2. La aplicación se hará en el paradigma funcional.
	2.3. Inicialmente, la aplicación consistirá de un solo archivo.
	2.4. Se harán funciones reutilizables para optimizar el programa.
